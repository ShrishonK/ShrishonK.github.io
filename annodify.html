Annodify

This project focuses on a Chrome Extension that helps write emails for you using a LLM. By using the Groq API and HTML injection, you can improve emails written in Gmail and Outlook using AI. You can select from Formal, Explanatory, and Concise. This extension aims to streamline the email writing process to help save time and improve email efficacy.
