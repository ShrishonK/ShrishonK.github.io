SummarAIze

This project is a summarizer for research articles that uses React, TailwindCSS, CSS, Chrome Extension, Groq LLM, and Firebase Hosting. This project utilizes a combined Chrome Extension that creates a summary of the article with customisability in terms of Tone and Length. This saves it to the Firebase database using an Express API and Firebase Functions.

#React #Firebase Database #Groq LLM API #Express JS #Google Chrome Extension #TeamPerformance
